# Analysis notebooks

Use this folder for team-authored analysis notebooks after the starter notebooks have been understood.

Suggested sequence:

```text
analysis/
  01-data-quality-and-alignment.ipynb
  02-geometry-target-extraction.ipynb
  03-thermal-and-sem-features.ipynb
  04-baseline-model-and-validation.ipynb
  05-uncertainty-and-error-analysis.ipynb
  06-final-results.ipynb
```

Each notebook should state its input data paths, use functions from `src/` where practical, and save final figures or metric tables to `results/`.
