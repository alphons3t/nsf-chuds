# Submission deliverables

This folder is the team’s handoff area. Keep editable source files in the relevant subfolder, then create the final ZIP in `submission/` immediately before uploading it to Qualtrics.

```text
deliverables/
  report/        # editable report source, figures, and final PDF
  presentation/  # editable slide deck and exported PDF
  submission/    # final upload checklist; ZIP files are ignored by Git
```

Before submission, verify that the ZIP contains the report PDF, executable notebook(s), presentation deck, and a text file containing the GitHub repository URL.
